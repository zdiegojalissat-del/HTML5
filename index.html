Comenzar

¿Qué es HTML5?


HTML5 es la versión más reciente del lenguaje HTML (HyperText Markup Language), utilizado para crear y estructurar páginas web. Este lenguaje define los elementos básicos de un sitio, como textos, imágenes, enlaces, formularios y otros contenidos que conforman la web.

¿Para qué sirve?
HTML5 permite:

  • Crear la estructura de una página web
  • Organizar el contenido (títulos, párrafos, listas, etc.)
  • Insertar imágenes, audio y video sin necesidad de plugins externos
  • Crear formularios para la captura de datos de los usuarios
  • Mejorar la compatibilidad con dispositivos móviles

¿Por qué es importante?
HTML5 es fundamental porque:

  • Es la base de todas las páginas web
  • Permite crear sitios más modernos, rápidos y accesibles
  • Facilita el uso de etiquetas semánticas como header, section y article
  • Es compatible con todos los navegadores actuales
  • Es esencial para el aprendizaje del desarrollo web junto con CSS y JavaScript

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title>Mi página</title>
</head>
<body>
</body>
</html>

<!DOCTYPE html>
Indica al navegador que el documento está escrito en HTML5. Sirve para que el navegador interprete correctamente el código y evite errores.

<html>
Es la etiqueta raíz del documento. Todo el contenido de la página web va dentro de esta etiqueta.

<head>
Contiene información que no se muestra directamente al usuario, como el título, metadatos y enlaces a CSS.

<body>
Aquí va todo el contenido visible de la página, como textos, imágenes, botones y formularios.

Etiquetas principales

h1 - h6

Los títulos en HTML son etiquetas que se utilizan para definir encabezados o secciones dentro de una página web. Van desde <h1> hasta <h6>, donde <h1> es el más importante y <h6> el de menor importancia.

p

Se usa para escribir bloques de texto.

img

Se usa para insertar imágenes

src → ruta de la imagen alt → texto alternativo (importante para accesibilidad)

a

Sirve para crear links o hipervínculos.

href → dirección del enlace

button

Crea botones interactivos.

ul / ol / li

1. Lista desordenada → ul

Se usa cuando el orden NO importa

Aparecen con viñetas (•)

2. Lista ordenada → ol

Se usa cuando el orden SÍ importa

Aparecen con números (1, 2, 3)

3. Elemento de lista → li

Es cada elemento dentro de la lista

Siempre va dentro de ul o ol

HTML5 semántico

header

Espacio para que el alumno describa el encabezado.

nav

Espacio para que el alumno describa la navegación.

main

Espacio para que el alumno describa el contenido principal.

section

Espacio para que el alumno describa las secciones.

article

Espacio para que el alumno describa los artículos.

footer

Espacio para que el alumno describa el pie de página.
